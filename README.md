# React Google Map Testing with Juan

## Tech-Stack
- React 18
- Node 16
- npm 8

## Install

```shell
git clone https://github.com/frontend-er/react-google-map-marker
cd react-google-map-marker
npm ci
```

## Development

```shell
# Start Vue.js frontend
npm run dev
```

## Testing
```shell
# Start Unit Testing
npm test
```

### Build
```shell
npm run build
```
